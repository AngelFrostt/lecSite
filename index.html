Что такое filter

Метод filter() для массива создает копию заданного массива, в которой остаются только элементы, соответствующие условию.

Более простыми словами:
1) Берётся каждый элемент массива.
2) Проверяется, выполняется ли для него заданное условие (например, больше ли число 10).
3) Если условие выполняется, элемент добавляется в новый массив.
4) Если нет — он пропускается.
5) В итоге получается новый массив с отфильтрованными значениями.

Метод includes() для массива определяет, содержит ли массив определенное значение, которое было передано методу, возвращая true или false.

Более простыми словами:
1) Берётся массив.
2) Проверяется, есть ли в нём нужное значение
3) ЕЕсли значение найдено - результат true, если нет - false.
